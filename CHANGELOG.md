### 🎨 配色参考
```css
--spotify-green: #1DB954       /* 主色 */
--spotify-green-hover: #1ed760 /* 悬停色 */
--spotify-black: #191414       /* 主背景 */
--spotify-dark-gray: #121212   /* 次背景 */

```

### 📝 使用建议
1. 推荐在暗环境下使用，护眼效果更佳
2. 绿色元素为交互重点，自然引导用户操作
3. 深色背景突出内容，减少视觉干扰
